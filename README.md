# 📊 Poll Results Visualizer

An interactive web dashboard for analyzing poll and survey data, built with Python and Streamlit. Ideal for data analysts, business analysts, and researchers.

![Streamlit Dashboard](images/dashboard_screenshot.png)

## 🎯 Problem Statement
Organizations collect poll responses but struggle to quickly extract actionable insights. This tool automates cleaning, aggregation, and visualization, making results accessible to non‑technical stakeholders.

## ✨ Features
- **Synthetic Data Generator** – Creates realistic poll data with demographic biases.
- **Data Cleaning** – Handles missing values and standardizes text.
- **Overall Vote Share** – Bar and pie charts with percentages.
- **Demographic Breakdown** – Stacked bar charts by region, age group, or gender.
- **Trend Analysis** – Line chart of response volume over time.
- **Interactive Dashboard** – Filter and explore data in real time via Streamlit.

## 🛠️ Tech Stack
- **Python 3.8+**
- **Pandas** – data manipulation
- **NumPy** – numerical operations
- **Matplotlib / Seaborn** – static visualizations
- **Plotly** – interactive charts
- **Streamlit** – web dashboard
- **Faker** – synthetic data generation
